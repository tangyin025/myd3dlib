myd3dlib
====
A fast and lightweight Direct3D9 based game engine.

Features
----
Direct3D9 renderer<br>
Deferred lighting<br>
Asynchronous resource loading<br>
Octree scene management<br>
Terrain LOD<br>
Lua scripted UI and game logic<br>
MFC scene editor<br>
